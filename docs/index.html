Documentation

Opereon is an open source tool for automatic orchestration of resources in ICT infrastructure. It can be used to perform any administrative tasks on Unix-like hosts.

Opereon does not require any agent installed on the remote hosts, and uses internally an OpenSSH client to perform remote administrative tasks. This ensures that everything works exactly as it would when the user would perform the same tasks manually.
